
🍕 Pizza Sales Performance Analysis

An end-to-end data analysis project that leverages SQL for data extraction and analysis, and Microsoft Excel for interactive dashboard development. The project transforms raw sales data into actionable business insights through data cleaning, analysis, and visualization.

📖 Project Overview

This project analyzes pizza sales transactions to evaluate business performance, customer purchasing behavior, product popularity, and seasonal sales trends. The objective is to help stakeholders make informed decisions using an interactive Excel dashboard powered by SQL-driven analysis.

🎯 Project Objectives
Analyze overall sales performance and revenue.
Identify top-performing pizza categories and products.
Evaluate sales contribution by pizza size.
Discover monthly and weekly sales patterns.
Build an interactive Excel dashboard for business reporting.

🛠️ Tech Stack
Microsoft SQL Server (SQL) – Data querying and business analysis
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Dashboard Design
Conditional Formatting

📂 Project Workflow
Imported the pizza sales dataset.
Queried and analyzed the data using SQL.
Generated business metrics and summarized results.
Built pivot tables and pivot charts in Excel.
Designed an interactive dashboard using slicers and KPI cards.
Interpreted insights to support business decision-making.

🔍 Key Insights
Classic pizzas generated the highest overall revenue.
Large-sized pizzas were the most profitable size category.
A small number of products contributed a significant share of total revenue.
Sales peaked in July, indicating strong seasonal demand.
Fridays recorded the highest sales volume, making them the busiest business day.
